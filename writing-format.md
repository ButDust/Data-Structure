# Writing Format

Use this format for every chapter. Replace the bracketed placeholders with your own English notes.

````md
---
outline: deep
title: [Chapter Title]
---

# [Chapter Title]

## Overview
[Write the scope and goals of this chapter.]

## Concepts
- [Concept]: [Short explanation.]

## Operations
| Operation | Description | Time Complexity | Space Complexity |
| --- | --- | --- | --- |
| [Operation] | [Description] | [O(...)] | [O(...)] |

## Algorithm Steps
1. [Step]
2. [Step]
3. [Step]

## Code
```cpp
// Paste your implementation here.
```

## Common Mistakes
- [Mistake]: [Correction.]

## Review Questions
1. [Question]

## Summary
- [Key point]
````

## Code Block Format

Use fenced code blocks and put the language name after the opening fence.

````md
```cpp
// C++ code here.
```

```c
// C code here.
```

```java
// Java code here.
```

```python
# Python code here.
```
````
